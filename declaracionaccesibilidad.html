Compromiso con la Accesibilidad

MedlinePlus se compromete a garantizar la accesibilidad de su sitio web para todos los usuarios, incluyendo a aquellos con discapacidades. Este compromiso está en línea con el Real Decreto 1112/2018 y con el objetivo de ofrecer un entorno digital inclusivo.

Características de Accesibilidad Implementadas

  • Idioma Principal: El idioma principal de este sitio web es el español.
  • Etiquetado Semántico: Se ha utilizado un etiquetado semántico adecuado para facilitar la navegación y la comprensión del contenido.
  • Texto Alternativo: Todas las imágenes en el sitio web incluyen texto alternativo descriptivo para que los usuarios de lectores de pantalla puedan entender su contenido.
  • Mecanismos de Navegación: Se han implementado menús de navegación claros y accesibles que permiten a los usuarios moverse fácilmente por el sitio.
  • Contraste de Color: Se ha asegurado que el contraste entre el texto y el fondo cumpla con las pautas de accesibilidad.
  • Uso de Teclado: Todas las funcionalidades del sitio son accesibles mediante teclado, lo que permite la navegación sin necesidad de un ratón.

Conformidad

Este sitio web es parcialmente conforme con el nivel AA de las Pautas de Accesibilidad para el contenido Web (WCAG) en su versión 2.1, debido a las excepciones y a la falta de conformidad en los aspectos que se indican a continuación:

  • Algunos documentos PDF pueden no ser completamente accesibles.
  • Se están realizando esfuerzos continuos para mejorar la accesibilidad en todo el contenido.

Contacto

Si encuentra dificultades para acceder a alguna parte de nuestro sitio o si tiene sugerencias sobre cómo mejorar la accesibilidad, no dude en ponerse en contacto con nosotros.
